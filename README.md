# ✨ Features

- 📌 Report Lost Items
- 📌 Report Found Items
- 📷 Upload Images
- 🔍 Search Items
- ✏️ Edit Records
- 🗑 Delete Records
- ☁️ Amazon S3 Image Storage
- ⚡ AWS Lambda Backend
- 📱 Responsive Design
- 🚀 Amazon EC2 Deployment
- 🔄 GitHub Actions CI/CD
